# Quickstart

> AI Attribution: This project is fully generated by AI (architecture, code, tests, and documentation).

## Repository Model

This repository is a single repo containing:

- engine code
- editor/application code
- a built-in sample project

The repository root is not the sample-project root.

- Repo root: source, apps, tests, scripts, docs
- Sample project root: `samples/physics_sandbox_project`

The sandbox startup path defaults to the built-in sample project under `samples/physics_sandbox_project`.

## 1. Install Dependencies

MSYS2 UCRT64:

```bash
pacman -S --needed mingw-w64-ucrt-x86_64-gcc mingw-w64-ucrt-x86_64-make
```

## 2. Configure And Build

Using CMake:

```bash
cmake -S . -B Build_mingw -G "MinGW Makefiles" -DBUILD_SANDBOX=ON -DBUILD_TESTS=OFF
cmake --build Build_mingw --target physics_sandbox
```

Using Make:

```bash
make sandbox
```

## 3. Run The Sandbox

From the repository root:

```bash
./Build_mingw/bin/physics_sandbox.exe
```

The executable expects the built-in sample project to be available at:

```text
samples/physics_sandbox_project/
```

Key content paths are resolved from that sample-project root, including:

- `samples/physics_sandbox_project/Packages`
- `samples/physics_sandbox_project/Prefabs`
- `samples/physics_sandbox_project/ProjectSettings`
- `samples/physics_sandbox_project/Scenes`

## 4. Controls

- `Space`: play/pause
- `N`: single step
- `R`: reset current scene
- `F1-F9`: load preset scenes
- `[` / `]`: previous/next scene
- `1`: spawn circle at mouse
- `2`: spawn box at mouse
- `Delete`: remove selected body
- `C`: toggle contact points/normals
- `V`: toggle velocity vectors
- `X`: toggle body centers
- Left mouse: select and drag dynamic body
- In config modal: `Up/Down` select parameter, `Left/Right` adjust value
- Click `配置` to open config popup
- Click `使用说明` to open help popup

## 5. Optional UI Font Override

Edit `sandbox_ui.ini`:

```ini
font_path=assets/fonts/ui.ttf
```

You can also set a system font path, for example:

```ini
font_path=C:/Windows/Fonts/msyh.ttc
```

## 6. Packaging

Portable package:

```bash
powershell -NoProfile -ExecutionPolicy Bypass -File scripts/package/package.ps1 -Version 1.0.0
```

Standalone core package:

```bash
powershell -NoProfile -ExecutionPolicy Bypass -File scripts/package/package_physics_core.ps1 -BuildDir Build_mingw -Version 1.0.0
```
