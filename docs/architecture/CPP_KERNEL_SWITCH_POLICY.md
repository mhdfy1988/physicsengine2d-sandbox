# `PHYSICS_USE_CPP_KERNEL` 开关策略

更新时间：2026-03-05

## 当前策略
1. 默认值：`ON`（C++ 内核为默认路径）。
2. 保留目的：为阶段 A/B 过渡期提供快速回退通道与问题定位开关。

## 保留周期
1. 在阶段 A 完成后继续保留 1 个稳定里程碑版本。
2. 阶段 B 期间若无回退需求与阻断问题，进入弃用流程。

## 弃用与移除
1. 弃用条件：
   - 回归与等价测试稳定通过；
   - 基准性能满足门限；
   - 无线上/集成方依赖 `OFF` 路径。
2. 移除动作：
   - 删除 `OFF` 分支逻辑；
   - 清理构建脚本中的兼容注释与条件分支；
   - 在变更日志明确说明“开关已移除，C++ 内核固定启用”。

## 例外处理
1. 若在阶段 B/C 出现阻断性稳定性问题，可临时延长保留周期。
2. 延长期内需附带问题单与退出时间点，避免开关长期悬挂。
